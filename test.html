Functions and Modules

Functions are blocks of reusable code that perform specific tasks. They help organize code, improve readability, and reduce redundancy. Modules are files containing collections of functions, variables, and classes that can be imported and used in other Python programs.

Defining Functions

Functions are defined using the def keyword followed by the function name, parentheses for parameters, and a colon. The code block within the function is indented.

Code Example:
def greet(name):
  """Greets a person by name."""
  print("Hello,", name + "!")

greet("Alice")  # Output: Hello, Alice!

Function Parameters and Return Values

Functions can accept parameters as input and return values using the return statement.

Code Example:
def add_numbers(x, y):
  """Adds two numbers and returns the sum."""
  sum = x + y
  return sum

result = add_numbers(5, 3)  # result will be 8

Modules and Importing

Modules are files containing Python code that can be imported and used in other programs. The import statement is used to import modules.

Code Example:
# math_utils.py (module file)
def square(x):
  return x * x

# main.py (your program)
import math_utils

result = math_utils.square(4)  # result will be 16

Built-in Modules

Python comes with numerous built-in modules providing various functionalities. Some commonly used modules include:

  • math: Mathematical functions
  • random: Generating random numbers
  • os: Operating system interactions
  • datetime: Date and time manipulation
